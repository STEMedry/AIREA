This project demonstrates the power of GenAI in transforming teachers' instructional ideas into interactive, web-based simulations. These tools provide engaging, cross-disciplinary STEM learning experiences, incorporating drones to foster critical thinking, spatial reasoning, and hands-on learning in a cost-effective way.

Satellite Positioning

Explore GNSS concepts in a dynamic 2D environment with real-time satellite movements and drone positioning.

Pedagogical Approaches: Inquiry-based Learning, Scenario Visualization

Virtual Flight

Practice drone piloting in a 3D virtual space, mastering control dimensions and enhancing decision-making skills.

Pedagogical Approaches: Experiential Learning, Competency-based Learning

QuadPhysics

Experiment with physics concepts like weight, gravity, and propeller speed, simulating diverse planetary conditions.

Pedagogical Approaches: Discovery Learning, Reflective Visualization

Dronography AOV

Explore relationships between AOV, FOV, and altitude while virtually measuring real-world structures and areas.

Pedagogical Approaches: Computational Thinking, Scenario-based Learning

Code Simulation

Design, test, and refine autonomous drone navigation code in a 3D environment with real-time path previews.

Pedagogical Approaches: Problem-based Learning, Algorithm Design
